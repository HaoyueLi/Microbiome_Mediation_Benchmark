# Microbiome Mediation Project

This is the github repository for reproducing the simulations and data analysis performed in the paper "Detecting and Quantifying Mediation of Health Outcomes by Microbial Communities" (Li, Accorsi, et al., 2026).

- Scripts under the `simulation` folder were used to run simulations.

- The simulations here require the R package `SparseDOSSA`, which can be found [this companion software repo](https://github.com/biobakery/sparseDOSSA).

- The simulation parameters can be read in from `parameters.xlsx`. 

- Scripts under the `real-world` folder were used to run real data analysis.

- The data for MedDiet study is available at: https://www.nature.com/articles/s41591-020-01223-3#data-availability

- The Blueberry RCT is currently in review and - in the interim - the data are available by request by contacting the authors. 

- Scripts under the `plotting` folder were used to generate the analyses and figures in the paper.
